Tool for presenting projects for your hackathon

Based on the MEAN.io stack

### Development
Requires a running mongodb

1. Install npm packages
```bash
npm install
```

2. Install yeoman packages
```bash
cd client
npm install
```

2. Install compass
```bash
gem install compass
```

3. Install grunt
```bash
npm install -g grunt-cli
```

4. Start dev server port 3000
```bash
node server.js
```
